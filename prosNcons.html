Pros for using single webpage

  • easy access everywhere
  • no delay in updates
  • no sensible information downloaded
  • can be ready soon

Cons for using single webpage

  • prone to mistakes
  • limited functions

Pros for using a database

  • customized functions
  • capable of containing sensible information

Cons for using a database

  • may take some time to set it up
  • the data needs to be updated regularly
